# Project Overview

OrthoNow is a responsive healthcare landing page developed for a fictional orthopaedic clinic as part of the NAMOZA Developer Assignment.

The objective of this project is to improve consultation lead generation while implementing modern analytics using Google Tag Manager and Google Analytics 4.
